# 植物大战僵尸双人对战 - 设计文档

## 概述

一款双人同屏对战的 Web 游戏，一人扮演防守方（种植植物），一人扮演进攻方（派遣僵尸）。使用 HTML/CSS + Canvas 技术栈在浏览器中运行。

## 交互控制

| 角色 | 设备 | 操作方式 |
|------|------|----------|
| 防守方（植物） | 键鼠 | 鼠标点击选植物→放网格；键盘 1-7 快速选植物；阳光自动收集 |
| 进攻方（僵尸） | 手柄 | LB/RB 切换僵尸，↑/↓ 选行，A 部署 |

使用浏览器原生 Gamepad API，支持 Xbox/PS/Switch Pro 手柄。

## 技术架构

- **前端 UI**: HTML + CSS (Flexbox/Grid)
- **游戏渲染**: Canvas API
- **游戏逻辑**: 原生 JavaScript (requestAnimationFrame 游戏主循环)
- **无外部依赖**: 纯原生实现，零框架

### 文件结构

```
index.html       - 主入口，HTML 结构 + CSS 样式
game.js          - 游戏主循环、状态管理
canvas.js        - Canvas 渲染（网格、单位、子弹）
plants.js        - 植物类定义和逻辑
zombies.js       - 僵尸类定义和逻辑
projectiles.js   - 子弹系统

input.js         - 键盘/鼠标输入
ui.js            - UI 面板交互
constants.js     - 游戏常量（数值、配置）
```

## 游戏规则

### 流程

1. 准备阶段 5 秒
2. 对战阶段实时进行（3 分钟限时）
3. 双方同时操作，互不干扰

### 胜负条件

- **僵尸赢得条件**: 任意僵尸到达最左侧列
- **植物赢得条件**: 坚持 3 分钟未被突破

### 阳光经济

| 项目 | 防守方 | 进攻方 |
|------|--------|--------|
| 开局阳光 | 150 | 100 |
| 自然掉落 | 每 10s 掉 2 个（各自收集） | 同左 |
| 额外收入 | 向日葵每 8s 产 25 | 消灭植物得 25 |
| 收集方式 | 自动收集 | 自动收集 |

自然掉落的阳光根据位置自动归属：左侧区域归防守方，右侧区域归进攻方，中间区域各半。向日葵产的阳光只归防守方。所有阳光自动收集，无需任何手动操作。

### 攻击规则

- 僵尸攻击力 40，攻速 2s → 3 下啃死豌豆射手 (100HP)
- 豌豆射手攻击力 20，攻速 1.5s → 5 下杀死普通僵尸 (100HP)
- 双发射手 ×2 → 2.5 秒杀普通僵尸
- 坚果墙 (1000HP) → 扛 25 下僵尸啃咬
- 樱桃炸弹 (800) → 秒杀铁桶僵尸 (550)
- 土豆雷 (500) → 秒杀路障 (200) + 普通 (100)
