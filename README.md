# 🌱 植物大战僵尸 · 双人对战 🧟

> 同屏实时对战 —— 一人种植物，一人派僵尸

---

## 🎮 如何运行

```bash
# 用 Python 启动本地服务器
python -m http.server 8080

# 浏览器打开
http://localhost:8080
```

需要两个玩家在同一台电脑上操作。**必须通过 HTTP 服务器访问**（直接打开 HTML 文件会导致手柄 API 无法使用）。

---

## 👥 操作指南

### 🟢 防守方（植物）— 键鼠操作

| 操作 | 方式 |
|------|------|
| 选择植物 | 点击左侧植物栏 或 按键盘 `1` `2` `3` `4` `5` `6` `7` |
| 种植植物 | 鼠标点击草坪上的格子 |
| 取消选择 | 按 `Esc` |

> 提示：快捷键 `1`-`7` 对应植物栏从上到下的顺序。

### 🔴 进攻方（僵尸）— 键盘操作

| 操作 | 按键 |
|------|------|
| 选择行 | `↑` `↓` 方向键 |
| 切换僵尸种类 | `←` `→` 方向键 |
| 部署僵尸 | `Enter`（在当前行右侧生成） |

### 🎮 进攻方（僵尸）— 手柄操作

| 操作 | 按键 |
|------|------|
| 选择行 | D-pad `↑` `↓` 或左摇杆上下 |
| 切换僵尸种类 | D-pad `←` `→` 或 `LB` / `RB` |
| 部署僵尸 | `A` 键 |

支持 Xbox、PS、Switch Pro 等主流手柄（浏览器原生 Gamepad API）。

## ☀️ 阳光系统

- **开局**：防守方 150 阳光，进攻方 100 阳光
- **自然掉落**：每 10 秒掉 2 个阳光，根据掉落位置自动分配
  - 落在战场左侧 → 归防守方
  - 落在战场右侧 → 归进攻方
  - 落在中间 → 双方平分
- **向日葵**：每 8 秒产 25 阳光，只归防守方
- **消灭奖励**：进攻方每杀死一个植物得 25 阳光
- **自动收集**：阳光落地后自动分配给对应玩家，无需手动点击

## ⚔️ 游戏规则

| 项目 | 说明 |
|------|------|
| 准备阶段 | 5 秒倒计时 |
| 对战时长 | 3 分钟 |
| 防守方胜利 | 坚持 3 分钟未被突破 |
| 进攻方胜利 | 任意僵尸走到战场最左侧 |

---

## ⚖️ 平衡说明

- 豌豆射手 5 下打死普通僵尸，僵尸 3 下啃死豌豆射手
- 坚果墙可以扛 25 下僵尸啃咬
- 樱桃炸弹秒杀铁桶僵尸及范围内所有单位
- 土豆雷可秒杀路障僵尸和普通僵尸
- 栅栏僵尸的栅栏只能挡 200 伤害，破了之后变普通僵尸
- 寒冰射手减速 50%，持续 3 秒

## 💡 策略提示

**防守方：**
- 前期多种向日葵建立经济优势
- 坚果墙 + 豌豆射手是经典组合
- 樱桃炸弹和土豆雷用来应对危机

**进攻方：**
- 普通僵尸开路，铁桶僵尸扛线
- 撑杆跳僵尸可以突破坚果墙防线
- 读报僵尸狂暴后速度快，适合突击
- 注意分散行数进攻，不要集中在一行
