🎮 分屏交互设计 v2

🔵 防守方（植物）— 键鼠

🖱️ 鼠标

  • 点击植物栏选择植物
  • 点击网格格子放置
  • 点击阳光收集

⌨️ 键盘快捷键

  • 1-7 快速选植物
  • Esc 取消选择
  • Space 收集全屏阳光

🔴 进攻方（僵尸）— 手柄

🎮 手柄映射

选择僵尸
  • LB / RB 切换种类
  • ←/→ 方向键切换
选择行
  • ↑/↓ 方向键选行
  • 左摇杆上下
操作
  • A 部署僵尸
  • X 收集阳光
  • Start 暂停

当前选中行和僵尸种类在界面上高亮显示，方便进攻方看清状态

📱 操作分工总览

🌱 防守方

鼠标点击精准种植物 → 适合需要精确定位的操作

🧟 进攻方

手柄快速切换部署僵尸 → 适合快速反应的派兵操作

⚠️ 技术要求

使用 Gamepad API（浏览器原生支持）

支持 Xbox、PS、Switch Pro 等主流手柄

手柄连接后浏览器自动识别，无需安装驱动
